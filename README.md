# Todo-App
A React based app to make a todo-app

Feel free to fork the project 
Clone th project and install the equired dependencies
Go to terminal and run -
-> npm init
-> npm install react react-dom --save
-> npm install babel-loader babel-preset-es2015 babel-presetreact --save-dev
-> npm install webpack webpack-dev-server --save-dev
-> npm install react router --save
 
 To run the project go to Terminal and type "npm start"
